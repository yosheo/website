Form App with Power Apps, Power BI, and Excel Online

The Problem: The communication systems reports for a client were generated every weekday to ensure the integrity of each resource's systems. The process was being performed manually each morning, taking a decent amount time to gather the statuses from various systems.

The Solution: Build a form app with Power Apps. The key user generates this report each morning by pulling the statuses of each resource from a few different systems. Data source integration wasn't possible to do within the client's timeframe. The solution then became to implement a form app that could take in communications statuses for each resource, save it to an Excel sheet hosted on the company's Sharepoint website, and integrate charts from Power BI so users could better visualize this daily report within the app.
